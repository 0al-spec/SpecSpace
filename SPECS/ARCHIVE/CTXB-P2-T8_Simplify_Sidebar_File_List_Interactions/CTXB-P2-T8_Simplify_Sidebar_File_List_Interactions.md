# CTXB-P2-T8 — Simplify sidebar file list interactions

## Objective Summary

Remove the "Open" button from sidebar file entries — clicking anywhere on the file cell should open/select the file. Replace the "Delete" button with a compact trash icon or "X" symbol that triggers a confirmation dialog before deletion.

## Deliverables

1. File cell click handler that selects/opens the file.
2. Remove the "Open" button from each file entry.
3. Replace the "Delete" button with a compact trash/X icon.
4. Confirmation dialog before delete proceeds.

## Acceptance Tests

1. Clicking a file cell opens/selects that file without needing a separate "Open" button.
2. Delete is triggered by a compact icon that shows a confirmation dialog before proceeding.
3. No accidental deletions are possible.
4. All tests pass.
